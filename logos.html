For this assignment, we created different types of logos for our fictional entity. I created a multitude of text-based and image-based logos.

For text-based logos, I used the two fonts from the visual identity board with varying color combinations to design the logotypes. Additionaly, I used the streetwear font to create lettermark logos.

For image-based, we continued to use the symbols created for the visual identity board. I also created a few combination logos, utilizing the griffin symbol with the main font and pictorial symbols utilizing the lettermark logos. These pictorial symbols work well with the current round avatar design trend found on many social media websites.
